┌─────────────────────────────────────────────────────────────────┐
│   COSMIC-style window bindings on macOS  (skhd + yabai)         │
│   Modifier  rcmd  =  Right Command key                          │
└─────────────────────────────────────────────────────────────────┘

  FOCUS WINDOW
    rcmd + h / j / k / l           focus left / down / up / right
    rcmd + ← / ↓ / ↑ / →           (same, arrow keys)

  MOVE WINDOW
    rcmd + shift + h/j/k/l         swap with neighbor in direction
    rcmd + shift + ←/↓/↑/→         (same, arrow keys)
                                   h/l at the edge → prev/next space
                                   (next past last → new space)

  WINDOW ACTIONS
    rcmd + q                       close window (cmd+w to focused app)
    rcmd + f                       toggle fullscreen (zoom to screen)
    rcmd + m                       toggle zoom to parent container
    rcmd + g                       toggle float / tile (this window)
    rcmd + s                       toggle split orientation (h ↔ v)

  APP LAUNCHERS
    rcmd + t                       new Ghostty terminal window
    rcmd + b                       open Brave browser
    rcmd + return                  application search (cmd+space → Spotlight)

  LAYOUT  (whole space)
    rcmd + r                       rotate layout 90°
    rcmd + e                       balance / equalize window sizes
    rcmd + shift + x               mirror layout on x-axis
    rcmd + shift + y               mirror layout on y-axis

  SPACES  (workspaces)
    rcmd + 1 … 9                   focus space N
    rcmd + shift + 1 … 9           move window to space N
    rcmd + ctrl + h / l            focus prev / next space
    rcmd + ctrl + shift + h / l    move window to prev / next space
                                   (next past the last → new space)

  DISPLAYS  (monitors)
    rcmd + alt + h / l             focus prev / next display
    rcmd + alt + shift + h / l     move window to prev / next display

  HELP
    rcmd + c                       show this cheat sheet
    cheat                          (shell alias — same thing)

  Notes
    • Left Command keeps every normal macOS shortcut (⌘C/V/W/Q/Tab…).
    • Space- and display-focus need yabai with SIP partial-disable.
    • Reload skhd after editing:  skhd --reload

Press  q  to close.
